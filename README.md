# Setup Instructions

Node 18+ required

navigate to DriversBoard root folder, then run following command

## 1. Install dependencies

npm install

## 2. Start server
npm run dev

## 3. Seed campaigns

Open in browser:
http://localhost:3000/api/campaigns

## 4. Open frontend

Open:
http://localhost:3000/

## 5. Run unit tests

npm run test

======================================================================================================

# Test Setup Instructions

Node 18+ required

navigate to DriversBoard/tests folder, then run following command

## 1. Install dependencies

npm install playwright
npx playwright install

## 2. Start Tests
npm run test:e2e
or
npm run test:ui
